# 📚 Vocabulary Learning App

একটি আধুনিক এবং রেসপনসিভ শব্দভাণ্ডার শেখার অ্যাপ্লিকেশন। যেখানে ব্যবহারকারীরা বিভিন্ন লেসন অনুযায়ী ইংরেজি শব্দ, তাদের অর্থ, উচ্চারণ এবং সমার্থক শব্দ জানতে পারবেন।

## ✨ প্রধান বৈশিষ্ট্যসমূহ (Features)

* **লেসন ভিত্তিক নেভিগেশন:** বিভিন্ন লেভেল বা লেসন অনুযায়ী শব্দগুলো সাজানো।
* **ডাইনামিক ডাটা ফেচিং:** সরাসরি API থেকে লেসন এবং শব্দের তথ্য লোড করা।
* **শব্দ উচ্চারণ (Text-to-Speech):** মাইক্রোফোন আইকনে ক্লিক করে সরাসরি শব্দের সঠিক উচ্চারণ শোনার সুবিধা।
* **বিস্তারিত তথ্য (Modal View):** প্রতিটি শব্দের অর্থ, উদাহরণ বাক্য এবং সমার্থক শব্দগুলো দেখার জন্য মডাল উইন্ডো।
* **ইউজার ফ্রেন্ডলি ইন্টারফেস:** অ্যাক্টিভ লেসন চিহ্নিত করার জন্য বাটন হাইলাইট সিস্টেম।

## 🛠️ ব্যবহৃত প্রযুক্তি (Technologies Used)

* **HTML5 & CSS3** (Tailwind CSS & DaisyUI)
* **JavaScript (ES6+)**
* **API Integration:** Fetch API
* **Speech Synthesis API:** শব্দের উচ্চারণের জন্য।

---

## 🚀 যেভাবে এটি কাজ করে

১. **ডাটা লোড:** পেজ লোড হওয়ার সাথে সাথে `datafetch()` ফাংশনটি লেসন লিস্টগুলো নিয়ে আসে।
২. **লেসন নির্বাচন:** কোনো লেসনে ক্লিক করলে `showLevelButton(id)` ফাংশনটি ওই লেসনের নির্দিষ্ট শব্দগুলো লোড করে।
৩. **শব্দ বিস্তারিত:** তথ্য আইকনে ক্লিক করলে `wordDeleils(id)` ফাংশনটি মডালে বিস্তারিত তথ্য দেখায়।
৪. **উচ্চারণ:** ভলিউম আইকনে ক্লিক করলে `pronounceWord(word)` ফাংশনটি ব্রাউজারের স্পিচ ইঞ্জিন ব্যবহার করে শব্দটি পড়ে শোনায়।

---

## 📂 কোড স্ট্রাকচার

* `datafetch()`: প্রাথমিক লেসন লিস্ট নিয়ে আসার জন্য।
* `showLevelButton()`: নির্দিষ্ট লেসনের ডাটা ফিল্টার করার জন্য।
* `wordDeleils()`: পপ-আপ মডালে শব্দের বিস্তারিত দেখানোর জন্য।
* `pronounceWord()`: ব্রাউজার স্পিচ সিন্থেসিস ব্যবহারের জন্য।
